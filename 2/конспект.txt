Контейнеры
-гетрогенные - наполниенны разными типами
1)Pair можно создать массив из них
2)Tuple
Опеределены операторы сравнения
Сравниваются почленно
-гомогенные - наполненны одним типом
1)на основе массива
1.1 array - лежит на стеке (поэтому размер лежит на этапе компиляции) std::array<int, 5>
1.2 vector - лежит в куче
1.3 deque(2 вектора столкнули носами) - лежит в куче
Плюсы:
-Поиск элемента за О(n)
-доступ по имеющемуся индексу за О(1)
-добавление/удаление в конец за О(1) am
-добавление/удаление в начало за О(N) - vect
                                 O(1) - deque
-добавление/удаление в середину O(N)
2)списки - смотряд друг на друга с помощью указателей
2.1 list -двусвязный
2.2 forward_list - односвязный
Плюсы:
-добвление и удаление в середину О(1)
-... конец и начало O(1)
-доступ O(N)
3)На основе деревьев-слева меньше, справа больше
3.1 map
3.2 set
3.3 multiset
3.4 mulsimap
map - хранит пару
Плюсы:
-поиск за O(log N)
-доступ за O(log N)
-добавление/удаление за O(log N)
4)На основе хаш-таблиц
4.1 unordered_set
4.2 unordered_map
4.3 unordered_multiset
4.4 unordered_multimap
Плюсы:
все действия за O(1) am

В STL элементы инициализируются 0 кроме array.
